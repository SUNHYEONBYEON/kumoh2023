# 용어
* Costmap
```
정의 :
costmap은 로봇 탐색에서 사용되는 지도의 한 유형.
지도의 각 셀에는 비용이 할당되어 있으며, 이 비용은 로봇이 해당 셀을 통과하는 데 필요한 비용을 나타냄.
비용이 낮은 셀은 통과하기가 더 쉽고, 비용이 높은 셀은 통과하기가 더 어렵다는 의미.
costmap은 로봇이 탐색할 수 있는 영역을 결정하는 데 사용.
로봇은 costmap을 사용하여 최단 경로를 찾거나 장애물을 피할 수 있다.

종류 : 
costmap은 2D 또는 3D로 생성될 수 있습니다. 2D costmap은 평면 지형을 나타내고, 3D costmap은 3차원 지형을 나타냄

costmap을 생성하는 방법 :
 1. 지도 데이터에서 직접 생성: 지도 데이터에는 도로, 건물, 나무 등과 같은 장애물의 위치가 포함되어 있음 이 데이터를 사용하여 costmap을 생성할 수 있음.
 2. 센서 데이터에서 생성: 로봇은 센서를 사용하여 주변 환경을 감지할 수 있습니다. 이 데이터를 사용하여 costmap을 생성할 수 있음
 3. 수동으로 생성: 로봇 운영자가 직접 costmap을 생성할 수 있음

costmap은 로봇 탐색의 필수적인 도구로 로봇이 안전하고 효율적으로 탐색할 수 있도록 해줌.
```
